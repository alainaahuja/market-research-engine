# Market Research Engine

Initial scaffold for a quant strategy workbench focused on single-ticker research, backtesting, and strategy comparison.

## Planned scope

- Backtesting engine for daily-bar strategy evaluation
- Performance and risk metrics
- Strategy comparison across multiple signal types
- Streamlit UI for interactive analysis
- Future AI-assisted analysis layer

## Initial structure

```text
src/
  analytics/
  backtest/
tests/
```

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
